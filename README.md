# Recent_dev_spm




## Name
LLM Consensus on SPM–Target Protein Interactions.

## Description
This project evaluates the reliability of Small Molecule (SPM) – target protein interactions generated by multiple Large Language Models (LLMs).

Different LLMs often produce varying or inconsistent biological predictions. This project introduces a consensus-based validation approach, where interactions generated by independent LLM runs are compared and filtered to identify high-confidence results.

The system processes outputs from multiple sources (my LLM run and a teammate’s LLM run), standardizes the data, and identifies overlapping interactions. Interactions agreed upon by multiple models are treated as more reliable, while unique results are considered potential false positives.

Key Features:

- Multi-LLM output integration

- Data cleaning and standardization

- Consensus-based filtering

- Structured CSV output generation

- Reproducible notebook-based pipeline

## Usage
Step 1: Prepare Input Data

Place LLM outputs (from different sources) into a folder, for example:

data/raw/

Step 2: Run the Pipeline
jupyter notebook spm_extraction.ipynb
Step 3: Output

- Processed data → cleaned datasets

- Final results → CSV file (e.g., result_17-mar.csv)
